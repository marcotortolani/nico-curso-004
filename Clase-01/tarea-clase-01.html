¿Qué es un runtime?

Es un paquete de código que implementa recursos para que un programa pueda hacer uso de las funcionalidades necesarias para su propio funcionamiento.

Ejemplos de Runtime:

  • NodeJS - (es un Runtime para JavaScript)
  • Deno - (es un Runtime para JavaScript y TypeScript)
  • JRE (Java Runtime Enviroment) - (Runtime para JAVA)

¿Qué significa HTML?

HTML por sus siglas en inglés significa "HyperText Markup Language" que traducido al criollo quiere decir "Lenguaje de Marcado/Etiquetado de HiperTexto", bueno tan criollo no es pero básicamente significa que no es Texto liso y plano como este párrafo, es un lenguaje interpretado a través de Tags donde se escribe el contenido deseado pero separando sus partes por estos tags o etiquetas.

Así es como tendremos el título encerrado por los tags <h1>TITULO</h1> o por ejemplo un párrafo <p>TEXTO DEL PÁRRAFO QUE ESTAMOS REDACTANDO</p> de esta manera cada elemento será interpretado por el browser.

¿Por qué HTML no es un lenguaje de programación?

Porque no se aplica ninguna lógica en su escritura, no tiene variables, funciones ni tampoco controles de ciclo, es un lenguaje que sirve únicamente para que el browser pueda procesar e interpretar cada elemento.

¿Qué son las HTML entities?

Según MDN las HTML entities son un texto (string), o podriamos decir un código, que comienza con un ampersand (la cosa rara esta --> &) y finaliza con un punto y coma (el viejo y confiable --> ;)

Se usan comunmente para mostrar caracteres reservados que de otra forma serían interpretados como código HTML
